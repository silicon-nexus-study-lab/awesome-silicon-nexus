# Awesome Silicon Nexus

> Curated resources from the **Silicon Nexus | Circuits & Code** community — courses, papers, tools, and datasets, organized by field. Built by the community, for the community.

If you're in our [Discord](https://discord.gg/your-invite-here), this is the GitHub home of everything shared in `#resources`.

## How to contribute

This is the easiest possible first contribution — no coding required.

1. Fork this repo
2. Add your resource as one line under the right field section, using this format:
   ```
   - [Resource Name](https://link-to-it.com) — one-line description of why it's useful
   ```
3. Open a pull request

That's it. Keep descriptions short and honest — "why would someone click this" is the bar, not a full review.

**Guidelines:**
- One resource per line, alphabetical order isn't required but appreciated
- No paywalled resources unless clearly marked `(paid)`
- No duplicate links — check the section first
- Broken link? Open a PR removing it or replacing it — that counts as a contribution too
